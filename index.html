What I do

I develop stochastic diversification models and apply machine learning techniques to bridge theoretical and empirical studies. Below is an overview of my skill sets and the technologies I use or am familiar with:

R & Markdown

Major tool for data handling, visualization, statistical analyses, modelling and simulation. Expertise in R package development.

C & C++

Familiar with C/C++ and skilled in integrating C++ with R/Python to solve performance bottlenecks.

Python

Secondary tool for study. Experienced in popular machine learning frameworks, e.g., Sklearn, PyTorch and PyTorch Geometric.

HTML, CSS & Vanilla JS

Familiar with basic tools for responsive website development. Run websites for fun.

Bash, Server & HPC

Proficient with UNIX-like systems, experienced in using HPC for parallel computation and neural network training, and skilled in running cloud servers.

LaTeX & TikZ

Proficient with LaTeX. Expertise in TikZ based graphics and visualization.

GIS & Bioinformatics

Skilled in GIS-based spatial analyses and well-versed in common bioinformatics toolchains

MySQL & Selenium

Use MySQL for big data management. Familiar with web crawling with Selenium.
